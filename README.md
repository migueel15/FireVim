### Installation
```sh 
# Remove current nvim config
rm -rf $XDG_CONFIG_HOME/nvim
rm -rf .local/share/nvim/

# Install firevim config
git clone https://github.com/migueel15/FireVim.git $XDG_CONFIG_HOME/nvim
```
